bitcoin.config — Utilities for reading bitcoin configuration files¶

Utilities for reading bitcoin configuration files.

bitcoin.config.read_config_file(filename)¶

Read a simple '='-delimited config file. Raises IOError if unable to open file, or ValueError if an parse error occurs.

bitcoin.config.read_default_config()¶

Read bitcoin default configuration from the current user’s home directory.
